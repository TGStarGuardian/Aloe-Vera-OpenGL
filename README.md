This is a project in Computer Graphics course.

 3. "Basic" topics:
  - Shaders
  - Textures
  - Camera
  - Affine transformations
  - Basic Lighting, namely: Directional and Point Lights
  - Materials and Lighting Maps
  - Model Loading and Model Rendering
  - Depth Testing
